# Prior

## Install

- **CMake 3.20+**               вҖ” https://cmake.org/download/
- **C++17** (MSVC)              вҖ” https://visualstudio.microsoft.com/downloads/
- **SFML 3.x**                  вҖ” https://www.sfml-dev.org/download.php
- **OpenGL**                    вҖ” `opengl32` on Windows (linked by CMake)

## Build

```text
cmake -B build -DSFML_ROOT=C:/SFML
cmake --build build --config Release
```

## Run

```text
.\build\Release\run.exe
```
